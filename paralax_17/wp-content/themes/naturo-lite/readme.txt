How to setup this theme?

Theme Setup : After login to the WordPress dashboard go to [Appearance-> Theme] and activate Naturo Lite Theme.

1 - After activating the theme kindly visit Appearance > Customize and check following options available:

Note : After changing any option click on "Save & Publish" Button to reflect the changes.

 A : Logo - This option is available in Pro version.
 
 B : Site Identity - Set the Title and Tagline of the site from this area.
 
 C : Colors - With this option you can change background color and color scheme of the site. For changing background color click on background color option and for changing color scheme of site click on color scheme option.
 
 D : Header Image - With this option you can set header image for you current header. For adding image simply click on "Add new image" button choose and crop image and set header image. 
 
 E : Background Image - With this option you can set body background image for site. For adding background image simply click on     "Select Image" and choose image with available option like Background Repeat, Background Position, Background Attachment.
 
 F : Menu - Select which menu would you like to use in header.
 
 G : Static Frontpage - Setting to set either your latest posts or a static page as the front page of your site.
 
 H : Home Boxes - Select page from dropdown for displaying on home below slider. Select page for box one, Select page for box two,     Select page for box three, Select page for box four. If you want to hide all these boxes simply check the option "Hide Page     Boxes".
 
 I : Slider Settings - 
 
 	 Slider Image :: Under heading "Slide Image 1, Slide Image 2, Slide Image 3" simply choose option remove image and upload     new or simply change the image and set slide image. 
	
 	 Slider Title :: Under heading "Slide title 1, Slide title 2, Slide title 3" put slide title for each slide.
	
   	 Slider Description :: Under heading "Slide description 1, Slide description 2, Slide description 3" put slide      		     description for slide.
	
 	 Slider Link :: Under heading "Slide link 1, Slide link 2, Slide link 3" put slide title for slide.		
	
 J : Social Settings : With this option you can manage site footer social icons. Suppose if you want to hide facebook icon so     simply make this field blank.
 
 K : Footer Area : With this option you can manage site footer about us section.
 
 L : Contact Details : With this option you can manage site header and footer contact details.

Or for deep information Click on Documentation and Support.
